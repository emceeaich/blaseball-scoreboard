# Blaseball Scoreboard

A terminal dashboard for the cultural event 
of [Blaseball](https://blaseball.com/).

Original code by @jazzychad.

Maintained by @jazzychad and @emceeaich.

Running: 

- git clone this project
- npm install
- npm start
